# 🏦 VAT-Flow Bot v1.0

Solução Fullstack para automação de contabilidade de IVA (Portugal/Espanha).

## 🚀 Tecnologias
- **Node.js** & **Telegraf** (Telegram Bot)
- **Neon Postgres** (Banco de Dados Serverless)
- **Express** (Dashboard Web Localhost:3000)

## 📊 Funcionalidades
- **Cálculo de IVA (23%)**: Separação automática de Bruto, Base e Imposto.
- **Persistência Real-time**: Dados salvos instantaneamente no Neon.
- **Dashboard Web**: Visualização tabular das últimas despesas via navegador.
